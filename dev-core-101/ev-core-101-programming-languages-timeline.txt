1.  Машинный код: Самый низкий уровень, состоящий из двоичных инструкций, понятных процессору.Создан в 1940
2.  Ассемблер: Язык низкого уровня, который использует мнемоники для представления машинных команд. Позволяет работать с аппаратным обеспечением более удобно, чем машинный код. Создан в начале 1950 годов
3. FORTRAN: Один из первых высокоуровневых языков, ориентированный на научные и инженерные вычисления. Создан в 1957 году
4.  COBOL: Язык, созданный для бизнес-приложений. Он широко используется в банковских системах и управлении данными. Создан в 1959 году
5. C: Язык программирования общего назначения, известный своей производительностью и низким уровнем абстракции. Основной язык для системного программирования. Создан в 1970 годах
6. Java: Объектно-ориентированный язык, известный своей переносимостью благодаря виртуальной машине Java (JVM). Широко используется для разработки корпоративных приложений и мобильных приложений на Android. Создан в 1995 году
7. Python: Высокоабстрактный язык, известный своей простотой и читаемостью. Широко используется в веб-разработке, анализе данных, машинном обучении и автоматизации. Создан в 1991 году
8. JavaScript: Язык программирования, используемый в основном для создания интерактивных веб-страниц. Он работает в браузере и на стороне сервера с использованием Node.js. Создан в 1995 году.
